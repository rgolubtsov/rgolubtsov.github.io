Starting up Mojolicious apps

To start up Mojolicious applications in production mode, in the background (like a daemon), redirecting all the console output to nowhere, i.e. suppressing it completely, use the following commands:

$ sudo -uwww-data /var/www/master/abcd_magistracy_admission_decree/script/abcd_magistracy_admission_decree daemon -m production -l http://*:8080 > /dev/null 2>&1 &
$
$ sudo -uwww-data /var/www/results/abcd_stage1_entrance_exams_results/script/abcd_stage1_entrance_exams_results daemon -m production -l http://*:8081 > /dev/null 2>&1 &
$
$ sudo -uwww-data /var/www/stage-1/abcd_stage1_admission_decree/script/abcd_stage1_admission_decree daemon -m production -l http://*:8082 > /dev/null 2>&1 &

They will be listening on specified TCP ports (in this example — on ports 8080, 8081, 8082 respectively) and serving HTTP requests from any client.
